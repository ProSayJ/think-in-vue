Vue中的计算属性-Computed

注：只要触发Add to A 或者 Add to B中的任何一个button，addToA方法和addToB方法都会执行,这样很耗费性能。耗时和搜索的业务场景建议使用该属性

普通调用js方法 addToA()。全局所有的方法都会执行 Age + A = {{addToA()}}

普通调用js方法 addToB()。全局所有的方法都会执行Age + B = {{addToB()}}

计算属性调用 addToA_C。只会查找computed中的方法 Age + A = {{addToA_C}}

计算属性调用 addToB_C。只会查找computed中的方法 Age + B = {{addToB_C}}
